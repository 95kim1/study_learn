# BFS

- 시작 정점에서 모든 정점을 1번씩 방문

- dfs와는 다르게 최단 거리를 구할 수 있는 경우가 잇는데, 가중치가 1인 그래프가 그러한 경우다.

- BFS를 이용해 해결할 수 있는 문제의 조건

  1. 최소 비용 문제
  2. 간선의 가중치가 1
  3. 정점과 간선의 개수가 적어야 한다.(O(V+E)와 제한 시간을 고려)
  4. 비용과 가중치의 단위가 일치 (거리면 거리 시간이면 시간)
